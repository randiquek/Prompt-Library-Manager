# Prompt-Library-Manager


A simple, full-stack web application for managing and organizing AI prompts with admin authentication.

## Overview

Prompt Library Manager helps you store, organize, and manage your collection of AI prompts in one secure location. Perfect for content creators, developers, and anyone who frequently uses AI tools and wants to keep their best prompts organized.

## What It Does

- **Store Prompts** - Save your favorite AI prompts with titles, content, and categories
- **Secure Access** - Admin login system keeps your prompts private
- **Easy Management** - Create, edit, and delete prompts through a simple interface
- **Track Changes** - Automatic audit logging shows who changed what and when

## Who It's For

- Content creators who use AI writing tools
- Developers testing AI models
- Teams sharing prompt templates
- Anyone who wants to organize their AI prompts

## Tech Stack

**Backend:**
- Go (Golang)
- SQLite database
- JWT authentication

**Frontend:**
- React
- Vite
- Axios

## Features

- Admin authentication with JWT tokens  
- CRUD operations for prompts  
- Audit logging for all changes  
- RESTful API  
- Modern React UI
